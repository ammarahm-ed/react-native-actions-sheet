# Route

A single component in router navigation stack.

## `name`

A name for the route

| Type   | Required |
| ------ | -------- |
| string | true     |

## `component`

A component to render when navigating to this route

| Type            | Required |
| --------------- | -------- |
| React Component | true     |

## `params`

Initial params for the route

| Type | Required |
| ---- | -------- |
| any  | false    |
