# DraggableNodeOptions

## `hasRefreshControl`

Whether the attached Scrollable view has a refresh control or not.

| Type | Required |
| ---- | -------- |
| boolean  | false    |

## `refreshControlBoundary`

By default refresh control gesture will work in top 15% area of the ScrollView. You can set a different value here.
Accepts a value between 0-1.

| Type | Required |
| ---- | -------- |
| number (0-1)  | false    |
